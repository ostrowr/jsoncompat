# jsoncompat.com

Source for the public `jsoncompat` website and interactive documentation.

## Run locally

```bash
pnpm install
pnpm start
```

## Validate changes

```bash
pnpm check
pnpm test
pnpm build
```

`pnpm build` also emits the Slidev presentation into `public/deck` so it is served from `/deck/`.
