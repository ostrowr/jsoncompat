# jsoncompat (JavaScript/WebAssembly)

`jsoncompat` is a JSON Schema compatibility checker for evolving schemas.

This package provides WebAssembly bindings for JavaScript so you can:

- Check whether schema changes are backward-compatible for serializer/deserializer roles.
- Generate example JSON values for a schema.

For full documentation and examples, see:

- https://jsoncompat.com
- https://github.com/ostrowr/jsoncompat

## Installation

```bash
npm install jsoncompat
```
